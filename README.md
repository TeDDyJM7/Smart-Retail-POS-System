# Smart Retail POS & Inventory Management System 🚀💰

A lightweight, high-performance desktop POS and real-time inventory configuration application built for small-and-medium retail shops.

## ✨ Key Features
- **Interactive Billing System:** Easily add products by barcode, update counts, and dynamically view sums.
- **Dynamic Stock Verification:** Prevents overselling by calculating current cart levels against structural database thresholds.
- **Automated PDF Receipts:** Generates thermal-printer ready professional shopping bills instantly upon checkout using `ReportLab`.
- **Persistent Data Store:** Powered by an analytical local `SQLite3` database layer keeping core sales metrics perfectly synced.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **GUI Engine:** Tkinter
- **Database:** SQLite3
- **PDF Generation:** ReportLab
